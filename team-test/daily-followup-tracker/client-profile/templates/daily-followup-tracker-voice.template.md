# Voice Profile — Daily Follow-Up Tracker

> This file is captured by `setup` (Step 6) at first run, and refreshable via setup's refresh mode.
> Setup writes this file to `<workspace>/client-profile/daily-followup-tracker-voice.md` (the workspace copy is gitignored — only this template ships).
> The work skill's drafting agents read this file to make drafts sound like the user, not AI.

## Email voice

(captured at setup)

**Greetings:** _e.g., "Hi <name>", "Hey", no salutation, etc._

**Sign-offs:** _e.g., "Best,", "Thanks,", "—<Name>", no signoff_

**Sentence rhythm:** _short / paragraph-form / mixed_

**Formality:** _formal / casual / mixed (per recipient)_

**Common phrases:** _signature openers, closers_

**Signature template:** _the auto-appended block at the end of the user's emails, if any_

## Slack voice

(captured at setup)

**Lowercase tendency:** _yes / no_

**Emoji frequency:** _none / occasional / frequent — and which emojis_

**Average message length:** _one line / short paragraph_

**Conversational openers:** _hey, hi, yo, none_

**Punctuation level:** _full stops or no, single sentences or runs_

**Thread vs. new message:** _do they reply in-thread by default or post new messages_
